## Projects

### SKU Intelligence Platform — Dashboard
(https://github.com/Ribas55/Dashboard.git)


An interactive **Streamlit dashboard** for sales analysis, demand classification, and forecasting at **SKU level**, designed to support **inventory planning, production decisions, and demand forecasting improvement** in a multi-SKU environment.

This project demonstrates practical application of **time-series analysis, demand segmentation, and forecasting model evaluation** using real-world analytics workflows.

---

## Key Features

* Interactive **sales time-series dashboards**
* **ABC/XYZ demand classification**
* **Intermittency analysis** (ADI & CV²)
* Multiple **forecasting models** with performance comparison
* **Weighted ensemble forecasting**
* Automated **SKU-level insights and recommendations**

---

## Forecasting Methods Implemented

* Moving Average (SMA)
* Exponential Smoothing (SES)
* ARIMA
* Linear Regression
* XGBoost
* TSB (for intermittent demand)

Model performance evaluated using **MAE, RMSE, and MAPE**.

---

## Tech Stack

**Python**, Streamlit, pandas, numpy, statsmodels, scikit-learn, XGBoost, matplotlib, plotly

---

## Project Context

Built as part of a **demand forecasting and sales analytics project** focused on improving forecasting accuracy across a **multi-SKU product portfolio** through **classification-driven modeling**.
